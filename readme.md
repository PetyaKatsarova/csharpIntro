This repo is a product of 2 hrs intro to blazor(front end), csharp and .net(framework). First you need to prep your
environment, do necessary installs. All we had to do(the previous step was done for us) was run those in the linux terminal:

dotnet new blazor // create new project

dotnet build

dotnet run

dotnet clean

In the browser, take url from Properties/launchSettings.json: http://localhost:5120

The page/url i worked on was: http://localhost:5120/student.
